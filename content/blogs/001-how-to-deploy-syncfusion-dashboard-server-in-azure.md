We have simplified the Syncfusion Dashboard Server deployment in Azure. All you need is to purchase a new virtual machine in Azure and Syncfusion Dashboard Server works with the lowest plan too.

1. Login into [Azure Portal](https://portal.azure.com) with your credentials

2. Go to New and search the keyword `Syncfusion`. From the search results, select the `Syncfusion Dashboard Server` and proceed to create.

3. From the next screen, create the credentials for your Virtual Machine. If you want to use any existing resource group, choose the existing one or create a new resource group.

4. Select the VM plan based on your requirements.

5. In the next step, disable the Boot diagnostics and use default settings for other options.

6. In the last step, view the VM summary and proceed to create the VM.

That's all. It will take some minutes to complete the deployment and you can use Syncfusion Dashboard Server once it is deployed.
